## 当前系统性能基线

| 指标 | 数值 |
|------|------|
| Chunk 实体数 | ~2,237 |
| Draw Call | ~2,200+ |
| FPS 稳态 | ~170 |
| FPS 峰值 | ~247 |
| FPS 谷值（加载尖峰）| ~58 |
| 帧时间 | ~5.9ms（均值），~21.8ms（峰值）|
| **瓶颈定位** | **CPU 端 Draw Call 提交 + 网格生成** |
