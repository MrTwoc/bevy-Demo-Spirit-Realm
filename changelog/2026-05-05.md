## [Plan B] - 2026-05-05

### 新增

- **src/atlas.rs**（新建）
  - `AtlasSlot` 结构体：`col`、`row` 字段，定义 atlas 中子图位置
  - `AtlasSlot::uv()` 方法：计算 slot 对应的 UV 范围
  - `TILE_SIZE_PX`、`ATLAS_WIDTH_PX`、`ATLAS_HEIGHT_PX` 常量
  - `grass` / `dirt` / `stone` 子模块：每个方块 6 个面的 slot 常量定义

- **src/chunk.rs**
  - `BlockTexture` 枚举：`GrassTop`、`GrassSide`、`GrassBottom`、`DirtTop`、`DirtSide`、`Stone`
  - `BlockTexture::from_block_and_face()`：根据 BlockId + Face 方向返回对应纹理
  - `BlockTexture::atlas_slot()`：返回 atlas slot 供 UV 计算使用

### 修改

- **src/chunk.rs**
  - `generate_chunk_mesh`：返回值从 5 元组改为 4 元组（移除 `Vec<[f32; 4]>` 顶点颜色）
  - `face_quad`：新增 `block_id` 参数；UV 从 placeholder 改为 atlas 真实映射
  - `spawn_chunk_entity`：材质从 `base_color: WHITE`（顶点颜色）改为 `base_color_texture`
  - `spawn_initial_chunks`：新增 `asset_server` 参数，加载 `textures/array_texture.png`
  - 移除：`block_color_as_rgba`、`face_uvs`、`ATTRIBUTE_COLOR` 顶点颜色属性

- **src/chunk_dirty.rs**
  - `rebuild_dirty_chunks`：同步更新为纹理方案；材质改为 `base_color_texture`
  - 移除：`ATTRIBUTE_COLOR` 顶点颜色属性

- **src/main.rs**
  - 新增 `mod atlas;`

### 纹理 atlas 规格

- 文件：`assets/textures/array_texture.png`，尺寸 250×1000px，子图 32×32px
- UV 槽位映射：
  - grass_top → (0,0)，grass_side → (1,0)，grass_bottom → (0,1)
  - dirt_top/dirt_bottom → (0,1)，dirt_side → (1,1)
  - stone 所有面 → (2,0)

## [chunk_dirty] - 2026-05-05

### 新增

- **src/chunk_dirty.rs**（新建）
  - `DirtyChunk` 组件：零数据标记组件，加在哪个实体上哪个实体就重建
  - `mark_chunk_dirty(commands, entity)`：公开 API，标记脏区块待重建
  - `is_air_chunk(chunk_data)`：辅助函数，判断区块是否全空气（Empty || Uniform(0)）
  - `rebuild_dirty_chunks` 系统：每帧检测 dirty 实体 → 重建 mesh → 清除标记
  - `set_block_dirty(commands, entity, chunk_data, pos, new_id)`：一次性原子操作（修改 + 标记脏）

### 修改

- **src/main.rs**
  - 新增 `mod chunk_dirty;`
  - `rebuild_dirty_chunks` 系统注册到 Update 阶段

### 已知限制

- 旧 mesh/material 未显式从 `Assets` 移除，会造成轻微 GPU 内存积累
- `mark_block_dirty`（在 chunk.rs 中）暂未与 `set_block_dirty` 联动
- `ChunkMeshHandle` 组件追踪旧 handle 暂未实现

- 已拆分模块：main.rs、camera.rs、cube.rs、input.rs、chunk.rs、chunk_dirty.rs、chunk_wire_frame.rs
